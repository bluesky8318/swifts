===============================================================================
说明:

1, 本软件需要在JDK1.6下运行

2, 所有的需要修改的配置文件都包含在jar中,在tools目录下,执行ConfigTemplateTool -d ../lib/framework,../lib/apps  ,即可把所有已经部署的需要修改的配置文件释放到llama/etc,llama/etc/keel-beans,llama/etc/keel-ws-beans等目录下,实现一个简单的便捷部署方案


==============================================================================================================================================================


启动llama的时候，可以增加JVM参数，
如果是HotSpot，可以增加： -Xms2048m -Xmx2048m -Xmn300m -Xss128k -XX:+UseParallelGC -XX:ParallelGCThreads=4 -XX:+PrintGC  -XX:-HeapDumpOnOutOfMemoryError


==============================================================================================================================================================

jvm同系统时间的校准
如果出现jvm时间与系统时间不一致的时候，可以在llama-1.1.0目录下的start.sh中配置参数来调整，参数是
TIMEZONE="-Duser.timezone=GMT"（可以用GMT+n<小时数>来调整时差）
VMARGS="$ENCODE $TIMEZONE"
